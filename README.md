# Kami 卡密批量生成工具

## 项目介绍
Kami 是一款专为脚本售卖场景开发的卡密管理工具，核心功能包括批量生成卡密、修改卡密信息、删除无效卡密，同时支持卡密验证功能，可帮助脚本开发者快速实现付费脚本的授权管理与分发，简化脚本售卖过程中的卡密管控流程。

## 使用方法
本项目的核心用途是为脚本出售提供便捷的卡密管理方案，通过生成唯一卡密实现脚本授权，开发者可通过卡密验证功能校验用户权限，确保付费用户正常使用脚本，同时可灵活管理卡密生命周期（生成、修改、删除），提升脚本售卖的效率与规范性。

1. 运行工具，根据需求设置卡密生成数量，点击批量生成按钮，生成所需卡密，生成后可直接在工具内修改卡密信息或删除无效卡密；
2. 当用户使用脚本时，通过工具的验证功能输入卡密，校验通过后即可完成脚本授权，确保用户合法使用。
